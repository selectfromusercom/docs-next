필요

- nodejs 19.x 이상
- npm 9.x 이상

# 설치

`npm install selectfromuser`

# 버전 확인

`selectfromuser -v`

<Tip>
더 짧게 사용가능 `slt -v`  
</Tip>

# 로그인

`slt login`

# 프로젝트 파일 만들기

`slt init`

# 프로젝트 연결

`slt link`

# 실행

`slt`

<Note>로그인, 파일 만들기를 건너뛰어도 됨</Note>

# 다른 프로젝트 연결하기

`slt unlink`

# 로그아웃

`slt logout`

# 로그인 상태 확인

`slt whoami`
